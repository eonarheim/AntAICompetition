Requirements:

 * Python 2.7.7 - https://www.python.org/download/releases/2.7.7/
 * Pip or easy_install for python
 * "Requests" Module installed

   On mac/linux with pip installed:
      pip install requests
   Windows easy_install:
      C:\Python27\Scripts\easy_install.exe requests

 * To Run type "python SampleAgent.py" at the command line
